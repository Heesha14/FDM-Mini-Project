ASSOCIATION

Description

Association rule mining is a method to identify the hidden patterns of co-occurrences in a dataset, by discovering entities that appeared together oftentimes in a group. This technique is also known as market basket analysis which can identify the Products bought together in each transaction or characteristics possessed by each individual.

Techniques

Apriori Algorithm has been applied for this purpose as a technique which is an algorithm in data mining used for frequent pattern mining. Frequent pattern mining methods calculate the probabilities of occurrences of each pattern and filter them with a threshold probability value
